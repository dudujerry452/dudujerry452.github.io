|总字数:140|阅读时长:1分钟|浏览量:

有时候想在某台机器上写代码, 并推送到github仓库. 然而又不想在该机器上配置自己的账户密钥, 这时候只需要:

  1. 运行 ssh-keygen -t ed25519 -C "your_email@example.com" 得到密钥对

  2. 在github仓库界面选择settings->deploy keys->add ssh key 将生成的公钥复制进来

  4. 在仓库中键入git remote set origin git@名字A:仓库url.git
